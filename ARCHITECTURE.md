## ğŸŒ System Overview

The AI Voice Retention Agent is a **full-stack customer retention system** with:
- **Frontend Demo Application** (Port 3000) - Web UI for agent demonstrations
- **Backend API Server** (Port 3001) - RESTful API with AI integration
- **MongoDB Database** (Port 27017) - Persistent customer data storage
- **Databricks Foundation Models** - AI/LLM services (Claude, GPT, Gemini, Llama)

## ğŸ¯ Summary

### **Key Architectural Decisions**

1. **Separation of Concerns**
   - Frontend and Backend are completely independent
   - Communication via REST APIs only
   - Each can be deployed/scaled separately

2. **Flexible Data Storage**
   - MongoDB as primary database
   - In-memory mock data as fallback
   - Seamless switching between sources

3. **AI Integration**
   - Databricks Foundation Models for flexibility
   - Support for multiple LLMs (Claude, GPT, Gemini, Llama)
   - Whisper for accurate speech-to-text

4. **Session Management**
   - In-memory for simplicity (development)
   - Ready to move to Redis (production)
   - Timeout-based cleanup

5. **Security First**
   - PIN verification with actual customer data
   - MFA fallback option
   - Environment variable management
   - No sensitive data in logs

6. **Natural Voice Experience**
   - Speech formatting for natural TTS
   - Currency, date, percentage handling
   - Browser-based speech synthesis

7. **Bilingual Support**
   - English and Spanish fully supported
   - Dynamic system prompts per language
   - Customer language preference

8. **Scalable Design**
   - Ready for horizontal scaling
   - Stateless API design
   - Database-backed persistence
